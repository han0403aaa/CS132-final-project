# CS132-final-project
Number Guessing Battle is a terminal-based number guessing game. The computer chooses a random number between 1 and 100, and the player tries to guess it using hints and the current possible range.

## How to Run
Run the program in the terminal with Python:
num_guessing.py

## How to Play

Choose a game mode from the menu.

In Easy, Medium, and Hard Mode, the computer secretly picks a number between 1 and 100. The player enters guesses, and the program gives hints about whether to try a higher or lower number. The game ends when the player guesses correctly or runs out of attempts.

In Reverse Mode, the player picks a number between 1 and 100 without telling the computer. After each computer guess, enter H if the number is higher, L if the number is lower, or C if the computer guessed correctly.
